前言

本文属于抛砖引玉型文章，每一点均可深入探索，在此不展开；

本文的命令均基于Mac进行，Win下有差异，自行搜索解决；

假如遇到问题，给一些思路：

  • 保证网络顺畅可翻墙（或采用淘宝镜像）
  • Win下构建C++，npm install windows-build-tools（耗时几分钟到几个小时不等）
  • 遇到权限问题，sudo + 命令 或sudo chown -R ${whoami} ~/（掀桌子行为）
  • 请确保语法、路径、项目名正确
